📲 Install Folio
Your browser supports one-tap install
Once installed
Folio lives on your home screen like a native app. Works offline, remembers your books and exactly where you left off. No app store required.
iPhone — Safari
1. Open this URL in Safari (not Chrome)
2. Tap the Share button ↑ at the bottom
3. Scroll down and tap Add to Home Screen
4. Tap Add — done ✓
Android — Chrome
1. Tap the ⋮ menu in Chrome
2. Tap Add to Home Screen
3. Tap Add — done ✓
💡 Your URL is: once you deploy to GitHub Pages it will look like yourname.github.io/folio-app — bookmark it in Safari first, then Add to Home Screen.
